nginx windows 下安装和配置

分类： nginx 2012-10-13 09:07 716人阅读 评论(2) 收藏 举报
nginxwindowsserversslkillprotocols

下载地址：http://nginx.org/download/nginx-1.2.4.zip

下载后直接解压到你的硬盘上，我的是d:\nginx

启动nginx

cd d:

cd nginx-1.2.4

start nginx

查看任务任务管理器里面会有nginx的进程

在浏览器输入http://127.0.0.1 此时会出现nginx的欢迎界面，说明启动nginx成功。

Welcome to nginx!

If you see this page, the nginx web server is successfully installed and working. Further configuration is required.

For online documentation and support please refer to nginx.org.
Commercial support is available at nginx.com.

Thank you for using nginx.

其他操作：

nginx -s stop          // 停止nginx
nginx -s reload       // 重新加载配置文件
nginx -s quit          // 退出nginx

linux下：

启动nginx：

/安装目录/nginx -c 配置文件的目录，-c用于加载配置文件，默认是在 安装目录下conf/nginx.conf

推出或者停止nginx：

强制停止：kill -9 nginx

从容停止：kill -QUIT `cat nginx pid所在目录`

快速停止：kill -TERM  `cat nginx pid所在目录`或kill -INT  `cat nginx pid所在目录`

平滑重启：kill -HUA `cat nginx pid所在目录`（平滑重启需检查配置文件是否正确：/安装目录/nginx -t -c）
